# Parth Prajapati — Portfolio

Personal portfolio site of Parth Prajapati, Analytics Engineer (supply chain analytics,
AI & automation, applied AI, operations research).

Single self-contained static site — no build step, no dependencies.

## Local preview
```bash
python3 -m http.server 4321
# visit http://127.0.0.1:4321/
```
Or just open `index.html` in a browser.

## Deploy
Hosted as a static site on Vercel. Framework: Other · Build command: none ·
Output directory: root. Pushes to `main` deploy to production automatically.
